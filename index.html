Weather Station

This ASP.NET MVC application reads data from several large XML files that was handed out by Aarhus University. These XML files contains data regarding weather conditions, temperatures etc. that has been collected all around Denmark. The application generates different graphs and shows relevant statistics in a visually pleasing manner. Search functionality has also been implemented to quicky sort through the data set and generate new graphs based on the search criteria. Furthermore, a visual representation of the data is also shown through Google Maps.
